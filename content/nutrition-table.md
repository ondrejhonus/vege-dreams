Having enough of every nutrient isn't that simple when you dont eat meat, but its very possible.

| Nutrients | Best source |
| :--- | :--- |
| **Proteins** | Eggs, Tofu, Nuts, Tempeh, Yogurt, Cottage Cheese, Lentils, Chickpeas, Falafel |
| **Healthy Fats** | Avocados, Nuts, Chia Seeds, Olive Oil, Cheese, Yogurt, Eggs |
| **Fiber** | Vegetables, Fruits, Whole Bread / Pasta / Tortilla, Legumes, Nuts, Seeds |
| **Carbs** | Potatoes (+Sweet), Oats, Rice, Bread, Pasta, Granola, Cereal |
| **Iron** | Spinach, Lentils, Chickpeas, Pumpkin Seeds, Quinoa |
| **Calcium** | Dairy (Milk, Cheese, Yogurt), Tofu, Brocolli, Kale, Oranges |
| **Vitamin B12** | Dairy (Milk, Cheese, Yogurt), Eggs |
| **Vitamin D** | Plant Milks, Eggs, Mushrooms |
| **Omega-3** | Chia Seeds, Flaxseeds, Walnuts |
